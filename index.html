Git e Github

git init: cria um novo repositório Git

git clone: clona um repositório existente do Git

git add: adiciona arquivos ao índice do Git (staging)

git commit: cria um novo commit no repositório

git push: envia os commits locais para o repositório remoto

git pull: puxa os commits remotos para o repositório local

git branch: lista, cria ou exclui branches

git checkout: muda para outro branch ou restaura arquivos

git merge: junta dois branches

git status: exibe o estado atual do repositório

git log: exibe o histórico de commits

git diff: mostra as diferenças entre arquivos

git stash: guarda temporariamente as mudanças

git reset: desfaz alterações, revertendo commits ou arquivos

git revert: reverte um commit específico, criando um novo commit
